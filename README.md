# Rundown App

Webapplicatie voor het maken van radio- en tv-draaiboeken
met samenwerking, audio-assets en visuele planning.

## Stack
- Next.js (App Router) + TypeScript
- Postgres
- MinIO (S3-compatible)
- WebSockets (presence + locks)
- Docker + Docker Compose
- Reverse proxy: Caddy
